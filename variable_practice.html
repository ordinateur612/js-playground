메인으로

주의사항

같은 변수를 let을 사용해 여러 번 선언하면 에러가 발생한다.

참고 사항

  • 카멜 표기법 사용
  • 변수명에는 문자, 숫자, 달러 기호 $, 밑줄 _ 만 사용 가능
  • 첫 글자에 숫자 작성 불가
  • 대소문자 구분 >>> apple / Apple 는 서로 다른 변수
  • 예약어는 변수로 사용 불가 (let, return, class, function)
  • "use strict" 모드에서는 let 없이 변수 선언 가능하나, "use strict"를 사용한 경우 let으로 변수 먼저 선언 필수
  • 상수 변수 선언은 const, 대문자 사용
